### What I do

**Reliability & Observability** — SLOs, alerting, and incident response with **Grafana**, **Prometheus**, and **Vault**. I instrument systems so failures are visible before users feel them, and I treat toil as a bug to be automated away.

**Platform & Infrastructure** — production-grade systems built with **Docker**, **Kubernetes**, **Proxmox**, **Ansible**, **Terraform**, **Traefik**, and **NGINX**. Repeatable, declarative, and reproducible — from bare metal to cloud.

**Automation & Tooling** — I build the glue: **Go** for daemons and CLIs, **Python** for pipelines, and IaC for anything I'd otherwise do twice by hand.

### Selected work

| Project | What it is |
| --- | --- |
| **[AxiOS](https://github.com/Emre-Diricanli/AxiOS)** | An AI-native operating environment — a model-agnostic intelligence daemon (`axiosd`) in Go. Bring your own model. |
| **[AlphaCluster](https://github.com/Emre-Diricanli/AlphaCluster)** | On-prem homelab cloud on **Proxmox + Docker + Ansible** — enterprise patterns on commodity hardware. |
| **[ndscan](https://github.com/Emre-Diricanli/ndscan)** | A modern CLI wrapper around `nmap` that makes daily network discovery faster and more intuitive. |
| **[netscan](https://github.com/Emre-Diricanli/netscan)** | A full-stack network scanner — a Go backend with a web UI for discovering and inspecting hosts. |
